# MovieSync -  Movies & Series Streaming Platform

Author: **mrabhi2k3** | [GitHub](https://github.com/mrabhi2k3)

## 🎬 Features

### Core Functionality
- ✅ **TMDB & IMDB Integration** - Multiple API keys for reliable data fetching
- ✅ **vidsrc-embed.ru Streaming** - High-quality video playback
- ✅ **Episode Selector** - Easy navigation for TV shows with seasons/episodes
- ✅ **Auto-Next Episode** - Seamless binge-watching experience
- ✅ **Movie & TV Show Details** - Plot, cast, ratings, similar content
- ✅ **IMDB Integration** - Direct links to IMDB pages

### Content Sections
- 🎬 Home - Trending, popular, top rated
- 🎥 Movies - Popular, top rated, now playing, upcoming
- 📺 TV Shows - Popular, top rated, on air, airing today
- 🔥 Trending - Daily and weekly trends
- 🔍 Search - Real-time content search

## 🚀 Routes

- `/` - Home page
- `/movies` - Movies collection
- `/series` - TV shows collection
- `/trending` - Trending content
- `/about` - About page
- `/privacy` - Privacy policy
- `/dmca` - DMCA policy


## 🔧 Tech Stack

- HTML5
- CSS3 (Custom + Tailwind)
- Vanilla JavaScript
- TMDB API
- vidsrc-embed.ru API
- Font Awesome Icons
- PWA (Service Worker)
